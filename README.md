# SharedView-Practice-
뷰 하나를 여러 뷰에서 각자의 용도로 사용하는 방식을 구현한 프로젝트입니다.
## 설명
- AView의 뷰모델인 AViewModel, 그리고 BView의 뷰모델인 BViewModel 각각에 chosenInt가 있습니다.
- AView와 BView에서 중복으로 사용하는 뷰 SharedView에서는 각각의 chosenInt를 수정합니다.
